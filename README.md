## ğŸ¯ Usage

### Create Account
1. Click "Sign Up" tab
2. Enter your name, email, and password
3. Click "Create Account"
4. You'll be automatically logged in

### Manage Tasks
1. Click "Add Task" to create a new task
2. Fill in title and description
3. Select initial status
4. Click status badge to cycle through states
5. Hover over task card to edit or delete

### Filter Tasks
- Click filter buttons to view specific task statuses
- See real-time task counts for each status
